# CYO_project
Project Overview
The project is based on the creation of a Machine Learning model that can predict Customer Churn. Customer Churn happens when a customer stops using a service. Now being able to forecast this helps businesses to be proactive in implementing measures to retain customers and minimize revenue loss. The system integrates machine learning, deployed via a web application, and uses data analysis to predict churn in real-time, all based off of customer information.

Objectives
1.	Analyse customer data to determine churn patterns.
2.	Develop and test machine learning models to make predictions.
3.	Create a web application that is interactive with Dash.
4.	Deploy the finished application for real-world use.
   
Project Structure
/data --> Dataset and cleaned data
/src --> Preprocessing and feature engineering code
/models --> Trained machine learning models (.pkl)
/app --> Dash web application
/docs --> Report and documentation

Project Workflow
1.	Data Collection & Cleaning
2.	Exploratory Data Analysis (EDA)
3.	Data Preprocessing & Feature Engineering
4.	Model Development (Logistic Regression, Decision Tree)
5.	Model Optimization & Validation
6.	Web Application Development (Dash)
7.	Deployment (Render)

Technologies Used
•	Python
•	Pandas & NumPy
•	Scikit-learn
•	Dash
•	Gunicorn
•	Render (Deployment)
